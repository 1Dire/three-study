MeshStandardMaterial 주요 속성

color

재질의 기본 색상입니다. 기본값은 0xffffff (흰색)입니다.

material.color.set(0xff0000); // 빨간색

roughness

재질의 거칠기를 나타내는 값으로, 0(매끄러움)에서 1(거칠음) 사이의 값을 가집니다. 기본값은 1입니다.

material.roughness = 0.5;

metalness

재질의 금속성을 나타내는 값으로, 0(비금속)에서 1(금속) 사이의 값을 가집니다. 기본값은 0입니다.

material.metalness = 1;

map

재질의 색상을 정의하는 텍스처입니다.

const textureLoader = new THREE.TextureLoader();
material.map = textureLoader.load('/path/to/texture.jpg');

aoMap

앰비언트 오클루전 텍스처로, 조명 효과를 강화합니다.

material.aoMap = textureLoader.load('/path/to/aoMap.jpg');

aoMapIntensity

앰비언트 오클루전의 강도입니다. 기본값은 1입니다.

material.aoMapIntensity = 0.5;

envMap

환경 맵 텍스처로, 반사와 굴절 효과를 위해 사용됩니다.

material.envMap = textureLoader.load('/path/to/envMap.hdr');

normalMap

표면의 미세한 디테일을 표현하는 노멀 맵 텍스처입니다.

material.normalMap = textureLoader.load('/path/to/normalMap.jpg');

displacementMap

정점 위치를 변경하여 표면 디테일을 추가하는 텍스처입니다.

material.displacementMap = textureLoader.load('/path/to/displacementMap.jpg');
material.displacementScale = 0.1; // 변위 강도

emissive

재질의 자체 발광 색상입니다. 기본값은 0x000000 (검정색)입니다.

material.emissive.set(0x00ff00); // 녹색으로 발광

emissiveIntensity

발광 강도입니다. 기본값은 1입니다.

material.emissiveIntensity = 0.5;

emissiveMap

발광 효과를 정의하는 텍스처입니다.

material.emissiveMap = textureLoader.load('/path/to/emissiveMap.jpg');

metalnessMap

금속성을 정의하는 텍스처입니다.

material.metalnessMap = textureLoader.load('/path/to/metalnessMap.jpg');

roughnessMap

거칠기를 정의하는 텍스처입니다.

material.roughnessMap = textureLoader.load('/path/to/roughnessMap.jpg');

alphaMap

투명도를 정의하는 텍스처입니다.

material.alphaMap = textureLoader.load('/path/to/alphaMap.jpg');
material.transparent = true; // 투명도를 활성화

bumpMap

표면의 거칠기를 시뮬레이션하는 범프 맵 텍스처입니다.

material.bumpMap = textureLoader.load('/path/to/bumpMap.jpg');
material.bumpScale = 0.5; // 범프 강도

clearcoat

클리어 코트 레이어의 강도입니다. 기본값은 0입니다.

material.clearcoat = 1; // 클리어 코트 활성화

clearcoatRoughness

클리어 코트 레이어의 거칠기입니다. 기본값은 0입니다.

material.clearcoatRoughness = 0.5;
